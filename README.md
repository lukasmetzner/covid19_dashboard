### Setup
**Requirements**
```bash
pip3 install -r requirements.txt
```
**Download data**
```bash
python3 ./scripts/data_downloader.py <your_country> # The default is Germany, if not specified
```
**Start streamlit server**
```bash
streamlit run ./streamlit_dashboard/app.py
```
